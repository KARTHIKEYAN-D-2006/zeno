# Zeno

Firebase hosting is deployed at:
- https://zeno-39d68.web.app
- https://zeno-39d68.firebaseapp.com

Hosting is now set to serve `main/index.html`.

## How to run locally

The project contains a `home/` frontend.

- Serve it via live server or static server.
- If you want Node local backend, use `server.js`.

## Firebase hosting commands

```bash
firebase login
firebase deploy --only hosting
```

## Clean setup
